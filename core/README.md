# MapAble Core

MapAble Core is the platform hub and shared UI shell for disability support services.

## Routes

| URL | Location |
|-----|----------|
| `/core` | `app/core/` — platform hub |
| Shared UI | `components/core/` |
| Navigation config | `lib/core-ui/` |

## Local development

```bash
pnpm dev
```

Open [http://localhost:3000/core](http://localhost:3000/core).
