# MapAble Cursor Rules

## Product identity

MapAble is an accessibility-first ecosystem for people with disability, families, carers, providers, councils, venues, employers, and transport partners.

## Non-negotiables

- Accessibility comes first — WCAG 2.2 AA minimum
- Semantic HTML, visible focus states, no colour-only status
- Forms need visible labels and useful error messages
- Plain language throughout

## Safety and compliance

MapAble is not emergency services, legal advice, clinical advice, or an NDIS funding decision-maker. Call 000 in immediate danger.

## Privacy and consent

Sensitive data is private by default, role-based, and shared only with clear consent.

## Ecosystem architecture

Reuse: vertical registry (`lib/mapable/verticals.ts`), shared landing components, interest form, boundary notices, ecosystem links.
