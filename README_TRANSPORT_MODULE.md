# Transport module (Phase 3)

## Built
Detailed transport bookings, operator assignment, driver/vehicle assignment, suitability warnings.

## Routes
`/dashboard/transport`, `/api/transport/bookings`, `/provider/transport`, `/admin/transport`

## Privacy
Mobility aids shared only with consent (`transport.accessibility_share`).

## Limitations
No live GPS or route optimisation (`TRANSPORT_LIVE_TRACKING_ENABLED=false`).

## Phase 4
Driver app, live tracking, route planning.
